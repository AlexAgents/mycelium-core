# Модуль: NonceManager

Потокобезопасное отслеживание nonce для единственного аккаунта Ethereum.

- **Файл:** `src/core/nonce_manager.py`
- **Класс:** `NonceManager`

---

## Конструктор

### `__init__(w3: Web3, address: str)`
Инициализируется с экземпляром Web3 и адресом в формате checksum.

---

## Методы

### `get_next_nonce() -> int`
Возвращает следующий доступный nonce. При первом вызове получает значение из сети, в дальнейшем инкрементирует внутренний счётчик.
Потокобезопасность обеспечивается через `threading.Lock`.

### `sync() -> None`
Повторно запрашивает актуальный nonce из сети с учётом ожидающих транзакций. Вызывается после ошибок транзакций.

### `reset() -> None`
Сбрасывает внутреннее состояние. Используется при смене аккаунта или создании новой сессии.
