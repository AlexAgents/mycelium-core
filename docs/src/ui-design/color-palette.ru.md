# Цветовая палитра

Документ описывает полный набор цветовых токенов, используемых в светлой
и тёмной темах **MYCELIUM CORE**.

## Принципы дизайна

1. **Никакого чёрного в светлой теме.** Цвет `#000000` не используется
   нигде. Весь тёмный текст использует `#1f3a68` (глубокий синий)
   или `#3d5a80` (вторичный).

2. **Два оттенка зелёного — намеренно.** `#1a7f37` / `#3fb950` для
   бейджей и кнопок, `#2da44e` для Toast-уведомлений. Разные уровни
   яркости для контраста на разных фонах.

3. **Цвета Toast не зависят от темы.** Они заданы через inline
   `setStyleSheet()` в Python-коде, а не через QSS-файлы.

4. **StatusBadge обновляются при смене темы.** `MainWindow` вызывает
   `refresh_theme()` на всех дочерних `StatusBadge` после применения
   новой QSS-темы.
