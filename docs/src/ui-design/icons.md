# Icon Library

All `qtawesome` icons used in **MYCELIUM CORE** with their semantic
meaning and location in code.

---

## Icon Source

All icons come from [qtawesome](https://github.com/spyder-ide/qtawesome)
using the **Font Awesome 5** icon set (prefix `fa5s.` for solid,
`fa5b.` for brands).

---
