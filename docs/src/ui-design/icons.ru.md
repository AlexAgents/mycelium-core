# Библиотека иконок

Все иконки `qtawesome`, используемые в **MYCELIUM CORE**, с их
семантическим значением и расположением в коде.

---

## Источник иконок

Все иконки из [qtawesome](https://github.com/spyder-ide/qtawesome)
на базе набора **Font Awesome 5** (префикс `fa5s.` — solid,
`fa5b.` — brands).
