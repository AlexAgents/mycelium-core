# Типографика

Шрифты, размеры и начертания, используемые в **MYCELIUM CORE**.

---

## Семейства шрифтов

| Контекст | Шрифт | Запасной | Использование |
|---|---|---|---|
| Текст UI | `Segoe UI` | `Ubuntu`, `sans-serif` | Все надписи, кнопки, заголовки, диалоги |
| Моноширинный | `Consolas` | `Courier New`, `monospace` | Адреса контрактов, TX hash, просмотр логов, поля ключей |

Оба шрифта предустановлены в Windows. На Linux/macOS автоматически
используются запасные варианты.

---

## Иерархия размеров

| Элемент | Размер | Начертание | QSS-селектор |
|---|---|---|---|
| Заголовок хедера | 14px | Bold | `#headerTitle` |
| Заголовок секции | 13px | Bold | `#sectionHeader` |
| Основной текст | 13px | Normal | `QWidget` (базовый) |
| Подписи полей | 11px | Bold | `#fieldLabel` |
| Значения статусов | 13px | Bold | `#statusValue` |
| Моноширинные поля | 12px | Normal | `#monoLabel` |
| Просмотр логов | 11px | Normal | `#logsViewer` |
| Мини-лог | 11px | Normal | `#logBox` |
| Статус-бар | 12px | Normal | `QStatusBar` |
| Подписи вкладок | 13px | 500 | `QTabBar::tab` |
| Текст тоста | 13px | 600 | `#toastContainer QLabel` |
| Текст бейджа | 11px | Bold | `StatusBadge` (inline) |
| Заголовок таблицы | 11px | Bold | `QHeaderView::section` |
| Тултип | наследуемый | Normal | `QToolTip` |

---

## Шкала начертаний

| Начертание | CSS-значение | Использование |
|---|---|---|
| Normal | 400 | Основной текст, поля ввода |
| Medium | 500 | Вкладки, стандартные кнопки |
| Semi-bold | 600 | Тост-уведомления |
| Bold | 700 | Заголовки, подписи полей, бейджи, кнопки действий |

---

## Межбуквенный интервал

| Элемент | Интервал | Назначение |
|---|---|---|
| Заголовок хедера | 2px | Акцент брендинга |
| Подписи полей | 1px | Читаемость прописных |
| Заголовки групбоксов | 2px | Визуальное разделение секций |
| Текст бейджа | 1px | Компактная читаемость |

---

## Межстрочный интервал

Используется стандартный Qt line-height. Единственное явное
переопределение — в текстовых лейблах `MessageDialog`:

```css
QLabel#messageDialogText {
    line-height: 140%;
}
```
