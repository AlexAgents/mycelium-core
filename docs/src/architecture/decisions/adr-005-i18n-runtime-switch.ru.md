# ADR-005: Локализация на лету

## Статус
Принято

## Дата
2026-05-20

## Контекст
Приложение должно поддерживать английский и русский языки. Перезапуск программы для применения нового языка ухудшает пользовательский опыт. Нам нужен способ мгновенной смены локали.

## Решение
Мы решили реализовать класс `I18N` (наследник `QObject`) с сигналом `languageChanged`. Каждая вкладка UI реализует метод `retranslate_ui()`. При смене языка `I18N` загружает новый словарь, генерирует сигнал, а `MainWindow` вызывает `retranslate_ui()` на всех вкладках.

## Последствия

### Положительные

- Бесшовный UX: язык меняется мгновенно без перезапуска и потери активной сессии.
- Централизованное хранение строк в `ru.json` и `en.json`.

### Отрицательные

- Требуется сохранять ссылки на виджеты (`self._lbl_*`) во вкладках для повторного применения текста во время работы.
