# MYCELIUM CORE

Desktop sandbox application for Ethereum-based voting simulation.

---

# Features

- Local Ethereum blockchain
- Smart-contract voting
- One vote per voter
- Whitelist enforcement
- Security audit subsystem
- QR vote receipts
- PyQt6 desktop UI
- Local sandbox environment

---

# Stack

- Python 3.11+
- PyQt6
- web3.py
- Solidity 0.8.20
- Geth

---

# Run

```bash
pip install -r requirements.txt
python main.py
